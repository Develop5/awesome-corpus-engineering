# Pregunta
¿Cómo puedo cambiar mi correo electrónico?

# Respuesta
Puedes cambiar tu correo desde el panel de usuario en la sección “Datos personales”. El sistema enviará un enlace de verificación al nuevo correo para confirmar el cambio.
