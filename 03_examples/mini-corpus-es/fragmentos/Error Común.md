# Error
El usuario no recibe el correo de verificación.

# Causa probable
El correo está en la carpeta de spam o el dominio del usuario bloquea mensajes automáticos.

# Solución
Pide al usuario que revise la carpeta de spam y marque el remitente como seguro. Si el problema persiste, solicita un correo alternativo.
