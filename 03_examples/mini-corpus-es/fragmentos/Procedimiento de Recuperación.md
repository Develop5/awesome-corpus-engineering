# Objetivo
Recuperar el acceso a una cuenta cuando el usuario ha olvidado la contraseña.

# Pasos
1. Accede a la página de inicio de sesión.
2. Selecciona “¿Has olvidado tu contraseña?”.
3. Introduce tu correo electrónico registrado.
4. Revisa tu bandeja de entrada y abre el enlace de recuperación.
5. Establece una nueva contraseña segura.
