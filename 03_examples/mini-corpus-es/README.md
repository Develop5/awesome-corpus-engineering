# Mini Corpus en Español — Corpus Studio

Este mini corpus es un ejemplo práctico de cómo estructurar documentación para sistemas de IA, chatbots empresariales y pipelines RAG. Incluye fragmentos claros, metadatos completos y una taxonomía inicial para demostrar buenas prácticas de corpus engineering en español.

## Contenido
- Fragmentos con metadatos
- Taxonomía inicial
- Ejemplos de FAQ, procedimientos y errores comunes
- Documentos fuente opcionales

## Objetivo
Servir como referencia pública para profesionales que desean aprender a documentar para IA de forma correcta y estructurada.

## Uso
Libre para estudio y análisis. No redistribuir como propio.
